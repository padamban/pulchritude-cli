# Install the CLI tooling

## Prerequisites

- Installed [Node.js](https://nodejs.org/en/download/)
- A node package manager

## Install the CLI tooling globally

### Install

Install it with your chosen package manager.

<Tabs>
<TabItem value="npm" label="npm">

```bash
npm install -g @pulchritude-cli/cli
```

</TabItem>
<TabItem value="yarn" label="yarn">

```bash
yarn add -g @pulchritude-cli/cli
```

</TabItem>
<TabItem value="pnpm" label="pnpm">

```bash
pnpm install -g @pulchritude-cli/cli
```

</TabItem>
</Tabs>

### Check the installed packege

```bash
CLI --version
```

## Install the CLI tooling into a folder

### Install

Create a folder for your CLI tool and init a node package.

Install the CLI tooling it with your chosen package manager.

<Tabs>
<TabItem value="npm" label="npm">

```bash
npm install @pulchritude-cli/cli
```

</TabItem>
<TabItem value="yarn" label="yarn">

```bash
yarn add @pulchritude-cli/cli
```

</TabItem>
<TabItem value="pnpm" label="pnpm">

```bash
pnpm install @pulchritude-cli/cli
```

</TabItem>
</Tabs>

### Register the script command

```ts title="package.json"
{
  "name": "cli-demo",
  "version": "1.0.0",
  "scripts": {
    // highlight-start
    "my-cli": "CLI"
    // highlight-end
  },
  "dependencies": {
    "@pulchritude-cli/cli": "latest"
  },
}
```

### Check the installation

Check if the CLI command is installed by running it with the version flag.

<Tabs>
<TabItem value="npm" label="npm">

```bash
npm run my-cli -- --version
```

</TabItem>
<TabItem value="yarn" label="yarn">

```bash
yarn my-cli --version
```

</TabItem>
<TabItem value="pnpm" label="pnpm">

```bash
pnpm my-cli --version
```

</TabItem>
</Tabs>
